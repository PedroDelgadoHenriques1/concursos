# Estudos — Concurso Cambé 001/2026 · Assistente Administrativo III

Material de estudo para o Concurso Público da Prefeitura de Cambé (PR), edital **001/2026**, cargo **Assistente Administrativo III**. Banca: **Instituto UniFil**.

## Conteúdo

| Arquivo | O que é |
|---|---|
| `index.html` | **Página inicial** — reúne os 3 materiais. Comece por aqui. |
| `apostila.html` | Apostila de apoio com a teoria do conteúdo programático. |
| `simulados.html` | 3 simulados completos (60 questões cada) com correção, nota por matéria e explicações. |
| `redacao.html` | Treinador de redação (prova discursiva): temas prováveis, coletânea, contador de linhas 15-25 e checklist com os critérios do edital. |
| `plano-de-estudos.md` | Plano de estudos semanal (6 semanas) com estratégia por disciplina. |
| `editais/` | Editais oficiais em PDF (`edital-001-2026.pdf` e versão com cabeçalho). |
| `leis/` | Textos oficiais das leis federais cobradas (CF/88, Lei 14.133, 8.429, 12.527, 13.460, 13.709/LGPD, 8.159, LC 95). |

## Dados da prova

- **Prova:** 12/07/2026 (manhã)
- **Objetiva:** 60 questões (35 Específicos · 15 Português · 5 Matemática · 5 Informática) — mínimo 50/100
- **Discursiva (redação):** dissertação de 15 a 25 linhas, peso 50, mínimo 25. **Zerar elimina.**

## Como usar

Abra o `index.html` no Chrome/Safari e navegue pelos materiais a partir dele. Os arquivos `.html` são autossuficientes e adaptados para celular. O progresso dos simulados fica salvo no navegador (localStorage), então abra sempre o mesmo arquivo/endereço para o salvamento funcionar.

**No iPhone:** abra a página no Safari → botão Compartilhar → "Adicionar à Tela de Início" para abrir em tela cheia, como um app.

> ⚠️ Falta baixar o **Estatuto dos Servidores de Cambé** e a **Lei Orgânica do Município (Títulos I, II e III)** — conteúdo "da casa" que a banca cobra e ainda não está na pasta `leis/`.
